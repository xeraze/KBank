# Security & Data Access Policy

## Data Extraction Prohibited
Any form of automated data extraction, including but not limited to scraping, crawling, or harvesting of `database.json` or any other files in this repository, is strictly prohibited.

## Purpose of Public Access
This repository is public solely to support the functionality of the associated GitHub Pages website. Public access does not grant any rights to use, copy, or analyze the underlying data.

## Disclosure
If you find any security vulnerabilities, please do not disclose them publicly. Contact the repository owner directly.
